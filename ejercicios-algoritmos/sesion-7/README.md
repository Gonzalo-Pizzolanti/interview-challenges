Para correr los tests primero instalá las dependencias:

```bash
pnpm install
```

Y luego ejecutá los tests del ejercicio que quieras:

```bash
pnpm test suma-de-rangos
```

O ejecutá todos los tests juntos:

```bash
pnpm test
```
